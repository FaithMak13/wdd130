W05 - CSS Questions

What tool is provided to share the article? — A social sharing toolbar with buttons for platforms like Facebook, Twitter, email, etc.

What CSS layout technique builds the toolbar? — Flexbox (display: flex)

How does the toolbar stay fixed when scrolling? — CSS position: fixed, which keeps an element in place relative to the viewport regardless of scrolling.

My personal reflection: After reading How to Be Yourself While Sharing the Gospel Online, I have come to the conclusion that sharing one’s faith is not something that requires any effort on the part of the person doing it. Instead, I came to understand that we can actually use our talents, experience, and personal connections in order to preach the good news to other people. One of such opportunities to share faith in a natural manner is using social media for communication.
